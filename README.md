# MoneyTokserver
express server deployed on Heroku for MoneyTok

> ## Technologies
> - Axios 
> - Cors
> - Crypto-js
> - Express
> - Express-session
> - mySQL
> - mySQL2
> - Node
> - Nodemailer
> - Passport
> - Sequelize

#### This is a split stack build with this repo deploying to Heroku, to be called from a separate repo containing the React front end deployed to GitPages.
